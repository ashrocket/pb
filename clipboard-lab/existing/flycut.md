# Flycut

## What it is
Flycut is a clean, simple clipboard manager for developers and an open-source
fork of Jumpcut.

## Clipboard features
| Feature | Status | Notes |
| --- | --- | --- |
| History depth | configurable | Reviewed source registers defaults of `rememberNum = 40` and `favoritesRememberNum = 40`, and the preferences UI exposes resizing of the remembered stack. |
| Search | yes | Reviewed macOS source wires a menu search box and reopens the menu specifically to make the search field work. |
| Image support | no documented image history | Reviewed macOS source declares `NSStringPboardType` on the main pasteboard and the help/save flows are text-oriented. |
| Pinning | favorites instead of pinning | Flycut has a separate favorites store and hotkeys to move items into it, rather than a generic pin/unpin surface. |
| Snippets | no | No snippet system was documented in the reviewed sources. |
| Sync | optional iCloud sync | Reviewed source includes `syncSettingsViaICloud` and `syncClippingsViaICloud` preferences through `MJCloudKitUserDefaultsSync`. |
| Hotkey | yes | README and help docs both describe the default main hotkey as `Shift` + `Command` + `V`, and the preferences allow it to be changed. |

## Storage
- Path: likely the app preferences plist under `~/Library/Preferences/`
  because reviewed source persists a `store` dictionary in `NSUserDefaults`;
  the exact bundle-specific plist path is not spelled out in the reviewed
  sources.
- Format: plist-backed `NSUserDefaults` store rather than SQLite.
- Encryption status: no at-rest encryption documented in the reviewed sources.

## Security posture
- Concealed/transient handling: yes
- Reviewed source defaults `skipPasswordFields` and `skipPboardTypes` to `YES`
  and includes `org.nspasteboard.TransientType`,
  `org.nspasteboard.ConcealedType`, `org.nspasteboard.AutoGeneratedType`, and
  `com.agilebits.onepassword` in the skip list.
- Password-manager exclusions: yes
- 1Password-specific pasteboard types are skipped by default.
- Networked: optional iCloud sync is present in the reviewed sources; no
  general-purpose web service or telemetry path was identified in the reviewed
  macOS code skim.

## Performance claims
- Startup time: unknown
- Memory: unknown
- Clipboard polling interval: 1 second; reviewed source creates a repeating
  `NSTimer` with `interval:(1.0)`.
- History size: default 40 for the main store and 40 for favorites.
- DB size limits: not applicable; reviewed source uses defaults/plist storage,
  not a database file.

## Known weaknesses
- Flycut is intentionally minimal and text-centric; reviewed macOS sources do
  not show first-class image history.
- The storage design is older and less auditable than a single SQLite file.
- At-rest encryption is not documented in the reviewed sources.
- The default 1-second polling interval is conservative and may feel less
  immediate than newer clipboard tools.
