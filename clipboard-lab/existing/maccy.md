# Maccy

## What it is
Maccy is a lightweight, keyboard-first macOS clipboard manager.

## Clipboard features
| Feature | Status | Notes |
| --- | --- | --- |
| History depth | configurable | Reviewed source enforces a user-configurable size limit; exact default is unknown. Public issue list shows a current bug report titled "Setting history size fails silently if above 999," so 999 appears to be a practical upper boundary today. |
| Search | yes | README documents type-to-search. |
| Image support | yes | Source accepts `.png` and `.tiff`; releases also mention OCR/search inside images. |
| Pinning | yes | README documents pin/unpin. |
| Snippets | no | No snippet feature is documented in reviewed sources. |
| Sync | no | No sync feature is documented in reviewed sources. |
| Hotkey | yes | Default popup shortcut is `Shift` + `Command` + `C`. |

## Storage
- Path: `~/Library/Application Support/Maccy/Storage.sqlite`
- Format: SQLite via SwiftData model storage
- Encryption status: unknown; reviewed sources describe Maccy as private/local but do not document at-rest encryption

## Security posture
- Concealed/transient handling: yes
- Default ignored types include `org.nspasteboard.TransientType`, `org.nspasteboard.ConcealedType`, and `org.nspasteboard.AutoGeneratedType`.
- Password-manager exclusions: yes
- Default ignored custom types include `com.agilebits.onepassword` and `net.antelle.keeweb`, plus several other app-specific pasteboard types.
- Networked: not fully documented; inference from the public source tree is that update-related networking likely exists because the repo ships `appcast.xml` and `SoftwareUpdater.swift`.

## Performance claims
- Startup time: unknown
- Memory: no published numeric claim found; README only claims "lightweight and fast"
- Clipboard polling interval: 500 ms by default; README documents lowering it to 100 ms
- DB size limits: no published DB size cap found; settings expose on-disk database size, and the practical history-size ceiling appears to be 999 from the current public issue tracker

## Known weaknesses
- Auto-paste depends on Accessibility permission.
- History-size behavior above 999 is currently reported as broken in the public issue tracker.
- No built-in snippets or sync were documented in the reviewed sources.
