This native macOS application is a broad power-user launcher whose clipboard history exists as a paid add-on capability rather than as part of the default free experience. Its design philosophy centers on search, automation, and extensibility first, with clipboard history joining that ecosystem only after the paid feature set is enabled. Once active, the clipboard component is mature and capable: it stores text, images, and file links, supports search, and lets users promote history items into a separate snippets and text-expansion workflow. Privacy defaults are conservative, with clipboard history disabled by default and concealed data ignored unless a user opts in. At the same time, some low-level implementation details remain undocumented, including the exact database path, storage engine specifics, and encryption-at-rest behavior. It reads as a configurable power tool rather than a clipboard-first specialist.
