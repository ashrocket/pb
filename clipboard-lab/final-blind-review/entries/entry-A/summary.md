This native macOS application is a clipboard specialist whose main design priority is improving local privacy without abandoning fast keyboard-driven use. Its standout choice is encrypted payload storage backed by key material in the macOS Keychain, paired with a broader settings surface for retention, polling, hotkey modifiers, excluded apps, and login behavior. Search is more expressive than simple keyword matching, supporting content type, source app, date, and encryption-state filters, while URL classification becomes more deliberate through data detectors. The security story is meaningfully stronger than plaintext-only local tools, but the design makes a visible tradeoff: searchable text stays in plaintext so substring search remains fast and convenient. This entry therefore feels like a pragmatic security upgrade, not a maximal lockdown design, and it still avoids sync, snippets, and broader workflow features.
