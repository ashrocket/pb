This native macOS application remains a focused, local-first clipboard manager, but shifts its design philosophy from bare-minimum correctness toward more practical daily use. It still keeps a fixed 200-item SQLite history and the same small menu bar footprint, yet adds explicit keyboard navigation, quick-pick shortcuts, PNG image handling, and live-reloaded privacy controls in a local config file. Search stays deliberately simple, using literal matching, but introduces a lightweight `image:` filter so image-heavy histories are easier to narrow. The overall product shape is still that of a specialist rather than a platform: there are no snippets, sync features, or workflow extensions. The biggest tradeoff is settings ergonomics. Privacy controls are materially better than the most minimal local entries, but they live in a watched config file rather than in a fully integrated preferences window.
