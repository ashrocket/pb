This native macOS application is a privacy-forward clipboard specialist that still aims to feel polished in daily use. Its design philosophy is local-only but ambitious: encrypted payload storage, a keyed search index that avoids plaintext searchable text in the database, on-device OCR for image clips, lazy loading, background image decoding, and a preview-oriented history panel all work together to make the tool feel more complete without turning it into a cloud service or workflow platform. Search becomes more structured and intentionally less permissive because encrypted token-prefix matching replaces arbitrary substring scans. That tradeoff is explicit and central to the product shape. The result is a focused clipboard manager with a real security story, richer image handling, and stronger perceived polish than the more minimal local-only entries, but still no snippets, sync, or generalized automation surface.
