This native macOS application is a focused, local-first clipboard manager that tries to do the core loop well without expanding into a broader productivity platform. The design philosophy is intentionally minimal: watch the general pasteboard, store recent items in a local SQLite database, and reopen them quickly from a floating menu bar window. It keeps a fixed 200-item history, captures plain text and images, uses one global hotkey, and offers straightforward substring search over stored metadata. Security choices are similarly explicit and narrow. Concealed and transient pasteboard types are skipped, the data directory is permission-hardened, and there is no network, sync, or update behavior in scope. That simplicity keeps the architecture small and inspectable, but it also leaves obvious tradeoffs: no settings UI, no pinning, no snippets, and no encryption at rest.
