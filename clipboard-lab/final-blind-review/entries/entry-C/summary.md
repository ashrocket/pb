This native macOS application keeps the same focused local-only shape as other specialist entries, but its design philosophy becomes more structured and desktop-like. Instead of relying on a hand-edited config file alone, it introduces a real settings window, scopes config watching more carefully, and expands search into a small query language with filters for source app, content type, and date. URL clips become a first-class type rather than being treated as generic text, which makes the search model more coherent. The underlying system remains deliberately simple in other ways: a permission-hardened SQLite database, a fixed 200-entry retention cap, local-only operation, and no snippets or sync features. It still stops short of at-rest encryption, but compared with more minimal clipboard specialists, this entry makes a stronger case for clarity, inspectability, and thoughtful keyboard-driven search.
