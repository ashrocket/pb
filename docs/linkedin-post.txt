Tired of selecting and copying text from your AI coding agent?

Same. I got so tired of highlighting a curl command, copying it, switching to Terminal, and pasting. Or grabbing a SQL query. Or a code block. Every single time.

So I built pb — a clipboard butter knife.

Say /pb. The agent finds the most recent useful artifact — query, command, code, URL, config — and puts it in your clipboard. No selecting. No highlighting. No dragging.

It's a Claude Code plugin, but the prompt is portable. Drop the same instructions into Cursor, Windsurf, Copilot — it works anywhere an agent can run a shell command.

Cross-platform too: macOS pbcopy, Linux xclip, WSL clip.exe. Auto-detects.

Sibling tool to agent-look (https://agent-look.raiteri.net — screenshot scanner) — same philosophy: stop dragging things around and let the agent handle the boring parts.
